# PlayNow

> Connecting people 

## About

PlayNow is a real-time recreational sports matching platform that helps users find nearby people interested in playing the same sport irl 

Instead of searching through WhatsApp groups or social media, users can instantly discover nearby players, receive a safe suggested public meeting spot, and enjoy their time to fullest.

Planning Phase

## Core Features
